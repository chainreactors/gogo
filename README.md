### 常见用法

扫描C段的关键端口

`./gt.exe  -k sangfor -ip 192.168.1.1/24 -p top2,win,db `

扫描启发式扫描B段或大于B段

`./gt.exe  -k sangfor -ip 172.16.1.1/12 -p top2,win,db,12345,10080-10081 -m s`

启发式扫描只会先扫描80端口,如果在该C段中扫描到了80端口,则进行已配置端口的完整扫描.加快扫描速度.

端口配置可以使用预设,单端口,和端口段随意组合.
